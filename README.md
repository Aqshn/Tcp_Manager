# TCP Connection Manager

## Overview
This project is a learning-focused application designed to explore TCP-based client-server communication and basic connection management.

It was created to understand how persistent connections work and how data can be exchanged reliably over TCP.

---

## Purpose
- Practice TCP socket programming
- Understand client/server architecture
- Learn connection handling and data transmission over TCP

---

## Technologies
- C#
- .NET
- TCP Sockets

---

## Functionality
- Establishes TCP connections between client and server
- Manages incoming and outgoing connections
- Sends and receives data over a reliable TCP channel

---

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Start the server application
4. Run the client and connect to the server using the configured address and port
